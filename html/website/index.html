Projects

This website is a portfolio of my projects; I am always working on something so this list is always expanding.
Raspberry Pi laser tripwire
This was a python project with the Raspberry pi. I built a laser tripwire using photocells, to detect brightness, and a laser. In response to the laser beam being broken, the Raspberry pi can communicate with other devices and trigger events such as turning on the lights or playing music.
Python mortgage model
This project uses the python library Pandas to analyze historical mortgage data. The model attempts to calculate the predicted interest rate of a mortgage based on historical data from the St.Louis FED and the historical data of stable interest loans.
Haskell bitcoin DNS seed lookup
This is the start of a project to build a bitcoin core full node implementation in haskell. So far I have created a DNS lookup to query bitcoin DNS seeds for the IP addresses of active nodes on the network.
